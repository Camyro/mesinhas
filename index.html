Sobre o projeto

O que é Mesinhas?

Uma coleção de micro-ferramentas e APIs da C1Studios. Cada ferramenta ocupa pouco espaço, mas serve bem ao seu propósito.

Como funciona?

A maioria roda direto no browser, sem servidor próprio. Usamos serviços de terceiros de forma transparente, com arquiteturas abertas e documentadas.

É grátis?

Sim. As ferramentas são gratuitas dentro dos limites dos provedores usados. Alguns recursos têm cotas mensais compartilhadas — o painel de cada ferramenta mostra o uso em tempo real.

Posso integrar em meu app?

Sim. Ferramentas com API ou TME Architecture funcionam em qualquer site, app web ou plataforma no-code como Kodular — via fetch, iframe ou WebViewer, sem autenticação.
