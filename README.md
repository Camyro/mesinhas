# Mesinhas

Pequenas ferramentas. Poder real.

Mesinhas é uma coleção de **micro-ferramentas online e APIs gratuitas** feitas para resolver problemas específicos de forma simples.
Sem instalação, sem cadastro — é só abrir e usar.

🌐 https://mesinhas.vercel.app/

---

## Ferramentas

**AnonMail**
Envie e-mails sem revelar sua identidade. Possui API e pode ser integrado em sites ou aplicativos.

**ImgConvert**
Converta imagens diretamente no navegador. O processamento é feito **100% localmente**, sem envio para servidores.

**TextSwap**
Substitua palavras ou trechos de texto em tempo real de forma rápida e simples.

---

## Em desenvolvimento

**AutoMail** – envio automático de e-mails para listas de contatos.
**TextMask** – anonimização automática de textos (nomes, e-mails e dados sensíveis).

---

## Como funciona

A maioria das ferramentas roda diretamente no navegador, priorizando:

* simplicidade
* velocidade
* privacidade

Algumas ferramentas utilizam APIs externas de forma transparente.

---

## Integração

Ferramentas com API podem ser usadas em:

* sites
* aplicações web
* plataformas no-code (ex: Kodular)
* WebView ou iframe
* requisições `fetch`

# Mesinhas

Small tools. Real power.

Mesinhas is a collection of **free online micro-tools and APIs** designed to solve specific problems quickly and simply.
No installation, no signup — just open and use.

🌐 https://mesinhas.vercel.app/

---

## Tools

**AnonMail**
Send emails without revealing your identity. Includes API support for integration in apps and websites.

**ImgConvert**
Convert images directly in your browser. Processing is **100% local**, with no server uploads.

**TextSwap**
Replace words or text fragments instantly.

---

## In development

**AutoMail** – automated bulk email sender.
**TextMask** – automatic text anonymization (names, emails and sensitive data).

---

## How it works

Most tools run directly in the browser, prioritizing:

* simplicity
* speed
* privacy

Some tools use external APIs transparently.

---

## Integration

Tools with APIs can be used in:

* websites
* web apps
* no-code platforms (e.g. Kodular)
* WebView or iframe
* `fetch` requests
